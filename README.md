## 📂 Featured Projects

### 🎓 [ResultHub](./ResultHub/README.md)
**Advanced Academic Data Processing Platform**
ResultHub is an end-to-end data ingestion, processing, and visualization platform serving thousands of engineering students. It features specialized ETL pipelines for OCR/PDF data, a high-speed search engine, and a dynamic student analytics dashboard.
- **Tech Stack:** Next.js 15, React 19, Tailwind CSS v4, Postgres, Redis, Python.

### 🏥 [Smilo](./Smilo/README.md)
**Comprehensive Clinic Management & AI-Powered Appointment Platform**
Smilo is an end-to-end digital clinic operating system. It features a scalable dynamic appointment engine, multi-branch management, and "Clio"—an embedded LLM AI assistant for seamless content management.
- **Tech Stack:** Next.js, Postgres (Prisma), NextAuth, Tailwind CSS, Mistral/Groq LLMs.

---

### ✈️ [TraveLedger](./TraveLedger/README.md)
**Complete Accounting & Financial Management Platform**
TraveLedger is a complete accounting and financial management web application built specifically for Travel Agencies, Tour Operators, and B2B Travel Agents. It handles everything a modern travel agency needs: tracking flight and hotel sales, managing payments, settling debts with suppliers, logging expenses, and providing a real-time view of business profits.
- **Tech Stack:** Next.js, Java Spring Boot, PostgreSQL, Tailwind CSS.
